# Hola, blog

Este es el primer post. El blog vive dentro del mismo repo del sitio, en
`content/blog/`, igual que los proyectos. Cada archivo `.md` es un post.

## Cómo escribo un post

1. Creo un archivo en `content/blog/`, p.ej. `2.mi-post.md`.
2. Le pongo el frontmatter (`title`, `slug`, `publishDate`, `description`, `tags`).
3. Escribo en markdown normal.
4. `git push` → Vercel despliega solo.

El número al inicio del nombre controla el orden en el listado.
